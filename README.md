# Hiring Pulse — India Tech Job Market Dashboard (2024–26)

An interactive analytics dashboard built with **HTML, CSS, JavaScript and Chart.js**,
analyzing 5,000 real job postings across 17 Indian cities and 15 industries.

---

## ✨ Features

- **Sidebar navigation** — 6 sections (Overview, Salary Insights, Skills Demand,
  Companies, Locations, Job Explorer)
- **4 live filter dropdowns** (Industry, Experience Level, Work Mode, Location Tier)
  that instantly recompute every KPI, chart and table row
- **Animated KPI counters** — count-up on load and on every filter change
- **14 Chart.js visualizations** — line, bar, horizontal bar and doughnut charts
- **Dark / light theme toggle** with saved preference (localStorage)
- **Searchable, sortable, paginated data table** with one-click CSV export of
  the current filtered view
- **Hover micro-interactions** throughout (cards, nav, buttons, table rows)
- **Fully responsive** — collapses to an off-canvas sidebar and single-column
  layout on mobile
- Built an interactive job-market analytics dashboard in HTML, CSS, JavaScript
  and Chart.js, processing 5,000 job postings across 17 cities and 15
  industries with zero backend dependency.
- Implemented dynamic client-side data pipeline (CSV parsing, filtering,
  aggregation) powering 14 live Chart.js visualizations and animated KPI
  counters that update in real time as users apply filters.
- Designed a fully responsive, dark/light-theme UI with sidebar navigation,
  a sortable/searchable paginated data table, and one-click CSV export of
  filtered results.
